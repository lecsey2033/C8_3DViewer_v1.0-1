Проект 3d Viewer

О программе

Разработана программа для визуализации каркасной модели в трехмерном пространстве. Программа работает с любыми файлами типа .obj.

Формат представления описаний трехмерных объектов .obj

Формат файла .obj - это простой формат данных, который представляет только трехмерную геометрию, а именно положение каждой вершины, положение UV координат текстуры каждой вершины, нормали вершин и грани, которые определяют каждый многоугольник как список вершин и вершин текстуры. Координаты obj не имеют единиц измерения, но файлы obj могут содержать информацию о масштабе в удобочитаемой строке комментариев.

Для того чтобы отобразить необходимо было заполнить структуру, которая содержала:

  • количество вершин
  • количество полигонов
  • 3d матрица для заполнения вершин x, y, z
  • массив полигонов

Программа может:

  • Отображать объекты
  • Перемещать объекты по осям x, y, z
  • Поворачивать объекты
  • Менять цвета вершин, полигонов, фона
  • Менять тип линии на сплошные и на
  • Приближать и отдалять объекты
  • Записывать gif изображения

Установка

Для того, чтобы начать использовать программу вам нужно в терминале использовать команду make install.

Удаление

Для удаления программы вы можете использовать команду make uninstall.
